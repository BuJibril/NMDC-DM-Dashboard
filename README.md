# NMDC HSE Executive Dashboard - Static Web Version

This version opens directly in any browser.

## How to use

1. Open `index.html`
2. Or upload the full folder to:
   - SharePoint
   - OneDrive public folder
   - Netlify
   - Vercel
   - Any web hosting server

## Notes

- No Python required.
- No server required.
- Added actions are saved in browser local storage.
- Export CSV and print/PDF are included.
